# auto-apt
AUTO-APT es es script en bash muy sencillo que automatiza algunos comandos de apt-get. Por ejemplo los comandos que mantiene el sistema actualizado, tales como apt-get update,apt-get upgrade etc..
Version: 0.2
Actualmente funciona en kali linux pero deber√≠a funcionar con cualquier derivado de Debian.. Su instalacion es muy simple. si tiene conocimientos sobre los comandos usted personalmente puede dar los permisos necesarios y moverlo a bin.

Instalacion:

sudo git clone https://github.com/John-W1ck/auto-apt.git

cd auto-apt

sudo chmod 777 instalar.sh

sudo ./instalar.sh

sudo auto-apt
